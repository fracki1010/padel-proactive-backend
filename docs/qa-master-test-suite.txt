# FORMATO:
#   ➡️  [ID] USER: <mensaje enviado>
#   ⬅️  [ID] BOT:  <respuesta esperada (o patrón clave)>
#   # comentario explicativo
#
# CONVENCIONES:
#   [PASS]  → debe responder con esa frase exacta o equivalente
#   [BLOCK] → el mensaje debe ser bloqueado/rechazado (cualquier rechazo vale)
#   [ANY]   → cualquier respuesta coherente es válida
#   [EMPTY] → el bot NO debe responder (mensaje ignorado)
#   (*) al final de un ID = test crítico — falla = regresión grave
#
# HOW TO RUN:
#   Enviar cada bloque a tu propio número de QA conectado al bot.
#   Reiniciar sesión entre bloques con "empezar de nuevo" o esperando TTL.
